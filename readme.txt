== Description ==

marble your wordpress is a widget display in the admin area and constantly remind you the power of marble and stons.
== Installation ==

This section describes how to install the plugin and get it working.



   1. Choose the right widget version for your wordpress version.
   2. Upload marble-your-wordpress.php to the /wp-content/plugins/ directory
   3. Activate the plugin through the 'Plugins' menu in WordPress
